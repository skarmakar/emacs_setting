Ubuntu
=============
```
mkdir ~/.emacs.d/
cp -r emacs_setting/emacs.d/* ~/.emacs.d/
sudo cp emacs_setting/etc_emacs/* /etc/emacs
cp emacs_setting/dot_emacs /home/{you username}/.emacs
```

Mac
============
```
mkdir ~/.emacs.d/
cp -r emacs_setting/emacs.d/* ~/.emacs.d/
sudo cp emacs_setting/etc_emacs/* /usr/share/emacs
cp emacs_setting/dot_emacs /home/{you username}/.emacs
```
